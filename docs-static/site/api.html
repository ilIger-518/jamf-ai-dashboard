Health

1 endpoint — all paths prefixed with /api/v1

GET /api/v1/health Returns 200 if the backend is alive

Auth

8 endpoints — all paths prefixed with /api/v1

POST /api/v1/auth/register Register a new local user account
POST /api/v1/auth/login Authenticate with username/password — returns access token and sets refresh cookie
POST /api/v1/auth/refresh Exchange refresh cookie for a fresh access token
POST /api/v1/auth/logout Invalidate the refresh token and clear the cookie
GET /api/v1/auth/me Return current authenticated user info
POST /api/v1/auth/change-password Change password for the currently authenticated user
GET /api/v1/auth/sso/microsoft/start Initiate Microsoft OIDC authorization flow
GET /api/v1/auth/sso/microsoft/callback Complete Microsoft OIDC exchange and issue session cookie

GET /api/v1/users/roles List all roles
POST /api/v1/users/roles Create a role (admin only)
PATCH /api/v1/users/roles/{role_id} Update a role (admin only)
DELETE /api/v1/users/roles/{role_id} Delete a role (admin only)
GET /api/v1/users/permissions List all available permission identifiers

Servers
