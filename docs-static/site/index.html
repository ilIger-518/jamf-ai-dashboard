Architecture

Core runtime components and dependencies.

Request and Sync Flow

Runtime path from UI interaction to Jamf and storage.

  1. User action in dashboard page triggers API call via frontend Axios client.
  2. Backend validates JWT and permission scope.
  3. Backend serves local data and/or fetches live Jamf data with OAuth client credentials.
  4. Sync jobs upsert entities into PostgreSQL and cache status in Redis.
  5. Knowledge and AI routes use ChromaDB and Ollama for retrieval and generation.
  6. Audit events are persisted to dashboard logs for settings visibility.
